SkillSetu Vocational Standard Operating Procedures (SOP)

====================================================
CARPENTRY SOP
====================================================

Timber Inspection Procedure

Before beginning any cutting or shaping operation, inspect the timber carefully.

Steps:
1. Check timber for cracks, knots, warping, or insect damage.
2. Ensure the timber is dry and suitable for structural work.
3. Confirm that the timber dimensions match the project requirements.
4. Reject timber pieces that show severe defects.

Best Practices:
- Use a straight edge to verify alignment.
- Avoid timber with large knots in load-bearing areas.


Measuring and Marking Timber

Accurate measurement and marking ensure correct carpentry work.

Steps:
1. Place the timber on a stable and level workbench.
2. Use a measuring tape or ruler to determine required dimensions.
3. Mark measurement points with a carpenter’s pencil.
4. Use a try square to draw straight lines across the timber.
5. Double-check all measurements before cutting.

Tools Used:
Measuring tape
Try square
Carpenter’s pencil
Marking gauge

Common Errors:
Incorrect measurement due to misaligned tape.
Cutting without verifying the marking line.


Cutting Timber

Steps:
1. Secure the timber using clamps or a workbench vise.
2. Align the cutting tool with the marked cutting line.
3. Begin cutting slowly to maintain accuracy.
4. Maintain steady pressure while sawing.
5. Smooth rough edges after cutting.

Tools Used:
Hand saw
Circular saw
Workbench clamps

Safety Notes:
Always wear safety goggles.
Keep hands clear of cutting blades.


====================================================
PLUMBING SOP
====================================================

Pipe Measurement and Cutting

Steps:
1. Measure the required pipe length using a measuring tape.
2. Mark the cutting point with a marker.
3. Secure the pipe in a pipe vise.
4. Cut the pipe using a pipe cutter or hacksaw.
5. Remove burrs using a deburring tool.

Tools Used:
Pipe cutter
Hacksaw
Measuring tape
Deburring tool

Common Mistakes:
Uneven pipe cuts.
Failure to remove burrs before installation.


Pipe Joint Sealing

Steps:
1. Clean pipe threads thoroughly.
2. Wrap thread seal tape around the threaded section.
3. Apply the tape in the direction of tightening.
4. Tighten pipe joints using an adjustable wrench.
5. Check for leaks after installation.

Tools Used:
Pipe wrench
Thread seal tape
Adjustable wrench


Leak Detection

Steps:
1. Turn on the water supply slowly.
2. Inspect all joints and fittings.
3. Look for dripping water or moisture buildup.
4. Tighten connections if necessary.
5. Replace faulty fittings if leaks continue.


====================================================
ELECTRICAL SOP
====================================================

Electrical Wiring Installation

Steps:
1. Turn off the main circuit breaker.
2. Verify absence of voltage using a voltage tester.
3. Strip insulation from wire ends using a wire stripper.
4. Connect wires using approved connectors.
5. Secure all connections inside junction boxes.

Tools Used:
Voltage tester
Wire stripper
Screwdriver
Wire connectors

Safety Notes:
Never work on live circuits.
Always use insulated tools.


Electrical Circuit Testing

Steps:
1. Turn on the power supply.
2. Use a multimeter to measure voltage.
3. Confirm correct voltage levels.
4. Inspect wiring for overheating or damage.
5. Document irregular readings if detected.

Tools Used:
Multimeter
Voltage tester


====================================================
WELDING SOP
====================================================

Surface Preparation

Steps:
1. Clean the metal surface to remove rust, oil, or dirt.
2. Use a wire brush or grinder if necessary.
3. Ensure surfaces are dry before welding.

Tools Used:
Wire brush
Angle grinder


Arc Welding Procedure

Steps:
1. Secure workpieces using clamps.
2. Adjust welding machine settings based on metal thickness.
3. Hold the electrode at the correct angle.
4. Strike the arc and maintain consistent arc length.
5. Move the electrode steadily along the joint.

Tools Used:
Welding machine
Electrode holder
Welding electrodes

Safety Notes:
Wear a welding helmet and protective gloves.
Ensure proper ventilation in the workspace.


====================================================
MASONRY SOP
====================================================

Bricklaying Procedure

Steps:
1. Prepare mortar using cement, sand, and water.
2. Spread mortar evenly on the base surface.
3. Place the brick on the mortar bed.
4. Press the brick firmly into position.
5. Use a spirit level to ensure alignment.

Tools Used:
Trowel
Spirit level
Masonry line


Mortar Mixing

Steps:
1. Combine cement and sand in the correct ratio.
2. Add water gradually while mixing.
3. Mix until a uniform consistency is achieved.
4. Use mortar immediately before it hardens.

Common Mistakes:
Adding excessive water which weakens mortar.
Improper mixing ratios.


====================================================
WORKSITE SAFETY PROCEDURES
====================================================

General Safety Rules

1. Always wear protective equipment.
2. Keep the work area clean and organized.
3. Inspect tools before use.
4. Replace damaged tools immediately.
5. Follow site safety regulations.

Personal Protective Equipment

Safety gloves
Safety goggles
Protective footwear
Construction helmet
