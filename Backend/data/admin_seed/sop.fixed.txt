**SkillSetu Platform**
Standard Operating Procedure (SOP)

1. **System Overview**

SkillSetu is an AI-powered vocational assessment platform designed to evaluate and enhance the competencies of blue-collar workers through automated voice interviews and multimodal AI evaluation.

The platform integrates several AI technologies including speech recognition, natural language processing, retrieval-augmented reasoning, and computer vision to evaluate practical skills and provide structured feedback.

The system performs the following core tasks:

• Conduct AI-driven voice interviews
• Evaluate responses using AI models
• Retrieve domain knowledge using SOP documents
• Analyze visual context through image inputs
• Generate automated feedback reports and analytics

The system architecture is divided into three primary components:

Frontend Client

Backend API Layer

AI Service Microservice

5. **Assessment Sequence Diagram**
User → Frontend: Start Assessment

Frontend → Backend: GET /api/assessment/categories

Backend → AI Service: Request category list

AI Service → Backend: Return categories

Backend → Frontend: Display categories

User → Frontend: Select category

Frontend → Backend: POST /api/assessment/start

Backend → AI Service: Initialize interview session

AI Service → Backend: session_id + first question

Backend → Frontend: Display prompt

User → Frontend: Record answer

Frontend → Backend: POST /api/assessment/{session_id}/answer

Backend → AI Service: Send audio + image

AI Service Processing:
   STT
   Translation
   SOP Retrieval
   Evaluation
   Counter Questions

AI Service → Backend: evaluation + next prompt

Backend → Frontend: Display feedback

Repeat until assessment completed

Backend → AI Service: GET summary

AI Service → Backend: Final evaluation

Backend → Frontend: Display report

6. **API Request Flow**
Assessment APIs
GET /api/assessment/categories

Returns available assessment categories.

POST /api/assessment/start

Starts a new interview session.

Response:

session_id
first_prompt
GET /api/assessment/{session_id}/next

Returns next prompt.

POST /api/assessment/{session_id}/answer

Request payload:

audio file
image frame

Response:

transcript
evaluation
feedback text
feedback audio
next prompt
GET /api/assessment/{session_id}/summary

Returns final evaluation report.

10. **Retrieval-Augmented Generation** (RAG) **Pipeline**

The system uses Retrieval-Augmented Generation to ground AI reasoning in SOP documents.

User Answer
      │
      ▼
Transcript Processing
      │
      ▼
Query Embedding
      │
      ▼
Vector Database Search
(Pinecone)
      │
      ▼
Top Relevant SOP Chunks
      │
      ▼
Context Assembly
      │
      ▼
Prompt Construction
      │
      ▼
LLM Reasoning
(Groq Llama / Gemini)
      │
      ▼
Evaluation Output
      │
      ▼
Skill Feedback Generation

Advantages of RAG:

• domain-specific reasoning
• reduced hallucination
• grounded procedural evaluation
